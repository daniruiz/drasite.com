DRA's dotfiles

Dotfiles are those files and directories found in a Unix-like OS that are used to save your system customizations. As the name implies, these files are named with a beginning dot, indicating they are special files, generally hidden by default in directory listing for regular users.

It is a pretty common practice to upload your Dotfiles to a public repository so that you can easily move them to other systems and get some ideas from others. So here you have mine!
