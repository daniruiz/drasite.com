Flat Remix GNOME theme

Flat Remix is a GNOME Shell theme inspired by material design. It is mostly flat using a colorful palette with some shadows, highlights, and gradients for some depth.

Installation

Manual installation

  1. Check the installed gnome-shell version: gnome-shell --version
  2. Download the corresponding theme for your shell version, and extract the compressed file.
    Download the appropriate theme for your version of gnome-shell [44, 43, 42, 40 & 41, 3.36 & 3.38, 3.34] ↧ Download Flat Remix
  3. Move "Flat-Remix-Blue-Light" folder to ".themes" in your home directory.
  4. Install user themes extension
  5. To set the theme, run the following command in terminal: 
    gsettings set org.gnome.shell.extensions.user-theme name "Flat-Remix-Blue-Light"
    or select "Flat-Remix" via gnome-tweaks.

Arch based distributions
  • flat-remix-gnome (AUR)

Login theme installation

Be sure you are capable of reinstalling gnome-shell/gdm in case it fails
You need to have installed imagemagick and glib-compile-resources:
  • Ubuntu
    sudo apt install libglib2.0-dev-bin imagemagick
  • Fedora
    sudo dnf install glib2-devel ImageMagick
  • Arch
    sudo pacman -S glib2 imagemagick
