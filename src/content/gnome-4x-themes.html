GNOME 4X themes

GNOME 4X Themes is a project dedicated to customizing the GNOME desktop environment starting from version 40. It offers custom colors and additional tweaks to enhance the user experience. The project supports themes for GTK-4, including Libadwaita, as well as GTK-3.

Features:

  • Dark & Light theme variants
  • 12 color variants
  • Rounded bottom corners hack for GTK-3 apps
  • Custom close/minimize/maximize buttons
