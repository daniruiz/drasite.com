Flat Remix ICON theme

Flat Remix is an icon theme inspired by material design. It is mostly flat using a colorful palette with some shadows, highlights, and gradients for some depth.




Themes
  • Flat Remix Dark - for dark interfaces
  • Flat Remix Light - for light interfaces

Installation

Manual installation

  1. Download and extract the compressed file.
  2. Move "Flat-Remix-Blue-Light" folder to ".icons" (./local/share/icons for KDE) in your home directory.
  3. To set the theme, run the following command in terminal: 
    gsettings set org.gnome.desktop.interface icon-theme "Flat-Remix-Blue-Light"
    or select "Flat-Remix-Blue" via your desktop specific tweaks application.

Ubuntu based distributions
sudo add-apt-repository ppa:daniruiz/flat-remix
sudo apt update
sudo apt install flat-remix
Fedora based distributions
sudo dnf install flat-remix-icon-theme
# To install all Flat-Remix themes run:
# sudo dnf install flat-remix-theme
